# Checkout51TestApp
The idea here is to implement a single activity app using Kotlin, Recycler view, JUnit and some sorting.

| Framework     | Libraries     | Networking    | Tests         |
| ------------- | ------------- | ------------- | ------------- |
|     MVP       | Kotlin        | Retrofit      | JUnit         |
|               | Recyclerview  | Picasso       | Mockito       |
|               | Cardview      | Moshi         |               |
|               | Lifecycle     |               |               |
|               | ConstraintLayout|             |               |
|               | Design        |               |               |
